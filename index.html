Mylyn ODA Data Source

An ODA Driver for Using Mylyn Tasks as ODA Data Source. Very useful for using Mylyn tasks and data as data source for BIRT reporting

Install Mylyn ODA Data Source

Point the following URL to your eclipse update site manager to install the Mylyn ODA Data Source:

Eclipse 4.3 (Kepler):

http://gzunino.github.io/mylyn-oda-datasource/p2/kepler
